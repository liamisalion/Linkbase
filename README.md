# LinkBase 常联系

LinkBase 常联系是一个面向职场个人的 AI 个人关系管理助手。它用于汇总邮件、日历、会议纪要、沟通记录和关注账号动态，帮助用户发现谁在等你、你在等谁、关系发生了什么变化，以及下一步该做什么。
